Student Management System
A simple yet robust Java console-based application for managing student records. Built to handle CRUD (Create, Read, Update, Delete) operations efficiently with data persistence.

✨ Features
Add Students: Register new students with ID, name, age, grade, and contact details

View Students: Display all students or search by ID/name

Update Records: Modify existing student information

Delete Students: Remove student records permanently

Data Persistence: Saves data to students.txt file for permanent storage

Search Functionality: Quick lookup by student ID or name

User-friendly Console Menu: Intuitive navigation with input validation

🛠️ Tech Stack
Language: Java (JDK 8+)

Build Tool: None (Pure Java)

Data Storage: File-based (TXT format)

IDE: IntelliJ IDEA / Eclipse / VS Code

🚀 Quick Start
Prerequisites
Java Development Kit (JDK 8 or higher)

Setup
Clone or download the project

Open terminal/command prompt in project directory

Compile: javac StudentManagementSystem.java

Run: java StudentManagementSystem
